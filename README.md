# Backofen-Simulation – Zweipunktregler

Interaktive Simulation zur Einführung in die **Regelungstechnik** im Unterricht der Elektrotechnik.

Simulation:
https://ulrike2.github.io/

## Ziel

Die Schüler*innen untersuchen spielerisch:

- den Unterschied zwischen Steuerung und Regelung
- den Aufbau eines Regelkreises
- die Funktionsweise eines Zweipunktreglers
- die Bedeutung einer Hysterese

## Funktionsweise

Die Aufgabe besteht darin, die Temperatur eines Backofens möglichst nahe am Sollwert von **200 °C** zu halten.

Dabei werden zentrale Begriffe der Regelungstechnik erarbeitet:

- **Regelgröße:** Temperatur
- **Stellgröße:** Heizung EIN/AUS
- **Störgröße:** z. B. Öffnen der Backofentür
- **Sollwert:** gewünschte Temperatur

Anschließend kann das Verhalten eines Zweipunktreglers mit und ohne Hysterese verglichen werden.

## Einsatz im Unterricht

Die Simulation wurde für eine Unterrichtseinheit zur Einführung in die Regelungstechnik entwickelt und ermöglicht einen handlungsorientierten Einstieg anhand eines Alltagsbeispiels.
